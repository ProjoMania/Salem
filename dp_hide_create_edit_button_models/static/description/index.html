Create/Edit Button restriction global or model wise

Features:

This app allows you to restrict the create/edit buttons based on user access group or based on models. You can restrict user to allow create/edit option or not. Or you can restrict the create,edit button model level.
Note: This will not work in many2one quick create, or kanban quick create. Note: This is view level visibility restriction only.

Flow

Select the users to set access rights.
