Vendor Sales Report

Configure if vendor needs to get sales report every week in partner form.

A mail with weekly vendor based sales report will be sent to all vendors enabled this option.

You can configure sales type in sales order form

One can manually export report form menu under sales reporting by providing a vendor.

User can simply export the report from selected date range using this menu.
