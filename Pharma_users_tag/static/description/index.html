Odoo 17 Pharma User Tag (Enterprise)

This module allows to add tags in user.

Note: This module is tested and working in Odoo Enterprise edition but not tested in Community version. OS.

Features

  • Allows to add tags in user.
  • When logged in as a user, only the sale orders, invoices, and deliveries of partners with matching tags to the user are visible.

How to use

  • Add tags to users.
  • Add tags in partners.
  • Create sale orders, invoices and deliveries.
  • 'In Settings > Users and Companies > Users, under Tags page, you can add tags.



  • Add tags to partners in 'Tags' field.


  • You can view only the sale orders that have partner tags matching those of the user.


  • You can view only the invoices that have partner tags matching those of the user.


  • You can view only the deliveries that have partner tags matching those of the user.
