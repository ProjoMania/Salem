Odoo 17 Pharma Payment Discount Approval


Note: This module is tested and working in Odoo in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.


How to use

  • Add payment discount in user.

  • If user add discount more than assigned discount then the user cannot create payments.


  • A button called 'Payment Discount Approval' is added to get approval. Another sales rep's permission will be needed to get discount more than discount added in user form.

  • After getting approval the user can get discount and can create payments.
