Odoo 17 Pharma Sale Expense (Enterprise)

This module allows to show hr expense in sale.

Note: This module is tested and working in Odoo Enterprise edition but not tested in Community version. OS.

Features

  • Show hr expense in sale order.
  • Show hr expense in invoice.
  • Allows to create expense from sale order and invoice.

How to use

  • In Invoicing > 'Expenses' button at top.
  • Click the 'Expenses' button and all expenses of current invoice will be listed. You can create new expense as well.
  • If you want to attach any expense to invoice, In Expenses > 'invoice' field, select invoice you want to attach the current expense with and create a new expense.
  • In Sales > 'Expenses' button will be shown if there is atleast one expense created fro the current sale order.
  • Note: In Invoicing, 'Expense' button is available for 'Posted' invoices only.
  • 'Expense' button at the top showing 0 expenses as right now there is no expense for this invoice

  • View after 'Expense' button click.


  • New Expense is created from current invoice. By default invoice is set in 'invoice' field


  • Create another new expense and select invoice you want to attach this expense with.


  • Now, expense count for invoice is changed to available number of expenses.

  • Now, on 'Expense' button click, you can see new listed expenses.
