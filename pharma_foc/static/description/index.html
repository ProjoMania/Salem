Odoo 17 Pharma FOC (Enterprise)

Provide free of cost products.

Note: This module is tested and working in Odoo Enterprise edition but not tested in Community version. OS.

Features

  • Allows the addition of free-of-cost (FOC) products to the main product.
  • When creating a sale order, you will receive the FOC product with a set percentage quantity of the main product at no additional cost.

How to use

  • Add the FOC product to the main product and set the allowable percentage quantity of the FOC product with the main product.
  • Create a sales order and include products that have FOC items.
  • Add FOC products of your choice. The quantity of these FOC products will be automatically set.
  • In Sales > Products > Products under FOC tab you can add FOC products and set quantity.


  • Create a sale order and add products.


  • Clicking on the 'Add FOC Product' button will open a window displaying a list of available FOC products.


  • Click on 'ADD' to include the FOC product in the order lines, with the quantity set as a percentage of the main product quantity.
  • These FOC items have been added at no additional cost.
