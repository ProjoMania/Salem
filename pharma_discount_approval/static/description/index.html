Odoo 17 Pharma Discount Approval (Enterprise)


Note: This module is tested and working in Odoo in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.


How to use

Discount in Sale Module

  • Add Maximum Discount in user form.

  • Have to enable discount button to show the discount button in sale order.

  • On Clicking button- Discount button below sale order line, the user can add global discount.

  • Discount Line is added, then it will check that if the discounted amount is greater than maximum discount or not.

  • If discount is more than assigned discount, if the user tries to confirm the quotation then it will show like below image.

  • Added a button- Approval for Discount to approve discount more than assigned discount.

  • If sales rep is login user then the same sales rep cannot allow to approve the discount.

  • On Clicking button- Approval for Discount then the user can get discount more than max discount.

  • After approving the discount user can confirm the quotation..


Discount in Invoice Module

  • On Clicking button- Discount button below invoice line, the user can add global discount.

  • Discount Line is added, then it will check that if the discounted amount is greater than maximum discount or not just like sale order.

  • If discount is more than assigned discount, if the user tries to confirm the invoice then it will show like below image.

  • Added a button- Approval for Discount to approve discount more than assigned discount.

  • If sales rep is login user then the same sales rep cannot allow to approve the discount.

  • On Clicking button- Approval for Discount then the user can get discount more than max discount.

  • After approving the discount user can confirm the invoice.
